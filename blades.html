Blades

addBlade() can create a blade without binding.

Text

Parameters | API 

const pane = new Pane();
pane.addBlade({
  view: 'text',
  label: 'name',
  parse: (v) => String(v),
  value: 'sketch-01',
});

List

Parameters | API 

const pane = new Pane();
pane.addBlade({
  view: 'list',
  label: 'scene',
  options: [
    {text: 'loading', value: 'LDG'},
    {text: 'menu', value: 'MNU'},
    {text: 'field', value: 'FLD'},
  ],
  value: 'LDG',
});

Slider

Parameters | API 

const pane = new Pane();
pane.addBlade({
  view: 'slider',
  label: 'brightness',
  min: 0,
  max: 1,
  value: 0.5,
});
